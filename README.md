# Vi-Link

A Discord bot project with modular architecture.

## Project Structure

```
vi-link/
├── server/          # Discord bot (TypeScript)
└── client/          # Web (TODO)
```

## Quick Start

### Discord Bot

See [server/README.md](./server/README.md) for setup instructions.

```bash
cd server
npm install
npm run dev
```

## Tech Stack

**Backend (Bot):**

- TypeScript
- Discord.js v14
- Express
- Docker
